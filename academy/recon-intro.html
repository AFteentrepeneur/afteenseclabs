Before any hacker touches a system, they do one thing first: Reconnaissance.
This is where the real game begins.

🔍 What is Reconnaissance?

Reconnaissance (or "recon") means gathering information about a target without being noticed. It's like being a detective before the crime even happens.

In cybersecurity, we call this OSINT — Open Source Intelligence. Everything is done using public information on the internet.

⚡ Why Does Recon Matter?

  • • Helps attackers find weak points (emails, subdomains, tech used)
  • • Makes phishing and social engineering attacks much stronger
  • • Ethical hackers use it to protect companies before real attacks happen
  • • Most real breaches start with good recon

🛠️ 5 Basic OSINT Techniques You’ll Use Today

1. Google Dorks
Special searches like site:novamobile.africa filetype:pdf
2. Social Media Search
Check LinkedIn, X, Facebook for employee posts
3. WHOIS & DNS Lookup
Find out who owns the domain and where it’s hosted
4. Public Documents
PDFs often leak names, emails, and software versions
5. Subdomain Tools
Free tools like crt.sh can reveal hidden parts of a website
